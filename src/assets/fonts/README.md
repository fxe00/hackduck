# 字体文件目录

请将你的字体文件放在这个目录中。

## 文件命名建议

### 主字体（界面文本）
- `Inter-Regular.woff2`
- `Inter-Bold.woff2`
- `Roboto-Regular.woff2`
- `Roboto-Bold.woff2`

### 代码字体（代码显示）
- `JetBrainsMono-Regular.woff2`
- `JetBrainsMono-Bold.woff2`
- `FiraCode-Regular.woff2`
- `FiraCode-Bold.woff2`
- `SourceCodePro-Regular.woff2`
- `SourceCodePro-Bold.woff2`

## 支持的字体格式

- `.woff2` (推荐) - 最小文件大小
- `.woff` - 兼容性更好
- `.ttf` - 最大兼容性

## 字体文件大小建议

- 单个字体文件 < 100KB
- 总字体文件 < 500KB
- 使用 `font-display: swap` 优化加载
